# userManagerPrivilegeSystem
基于Qt 的用户权限管理系统
采用C/S架构、mvc架构，前后端分离，模块化管理，使用模型视图框架、代理。登录模块采用JWT,后端模块使用QHttpServer，
使用Json传输数据，采用数据库连接池实现MySql连接复用和负载均衡，自定义登录控件。
